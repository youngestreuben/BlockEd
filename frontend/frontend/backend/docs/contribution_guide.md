# Contribution Guide for BlockEd

Thank you for contributing to **BlockEd**!

## Steps to Contribute
1. Fork this repository and clone it locally
2. Create a feature branch
3. Make changes (smart contracts, frontend, backend)
4. Test your changes
5. Commit & push your branch
6. Submit a pull request

## Guidelines
- Follow Solidity and JavaScript best practices
- Ensure smart contracts are secure and gas-efficient
- Document all changes
- Respect MIT open-source licensing
