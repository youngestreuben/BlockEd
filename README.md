# BlockEd: Blockchain for Children

**BlockEd** by XAYN TECHNOLOGY is a blockchain platform designed to **fund, track, and incentivize digital public goods (DPGs) for children**. The platform ensures transparency, accountability, and community governance.

---

## Features
- Transparent donation tracking (Donation.sol)
- ERC-20 token rewards for creators (IncentiveToken.sol)
- Decentralized governance and voting (Governance.sol)
- Fully open-source and modular
