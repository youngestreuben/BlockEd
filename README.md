# BlockEd

## Project Overview
BlockEd is an innovative platform designed to streamline the process of creating and managing educational content using a modular approach. The platform allows users to effortlessly combine different educational blocks to create interactive lessons.

## Features
- **Modular Design:** Easily combine and rearrange different educational blocks.
- **User-Friendly Interface:** Intuitive design for both educators and students.
- **Customizable Content:** Tailor educational materials to meet specific needs.
- **Collaboration Tools:** Work together with other educators in real-time.

## Setup Instructions
1. **Clone the Repository:**
   ```bash
   git clone https://github.com/youngestreuben/BlockEd.git
   cd BlockEd
   ```
2. **Install Dependencies:**
   ```bash
   npm install
   ```
3. **Run the Application:**
   ```bash
   npm start
   ```
4. **Access the Application:**  Open your web browser and go to `http://localhost:3000`.
